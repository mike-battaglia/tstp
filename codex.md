# Commonly Used Shortcodes

## Membership Based Restrictions
```php
// Memberships Content Restriction
[wcm_restrict plans="all-access, man-ed, con-ed, sales-tax-vip"]
Only members can see me.
[/wcm_restrict]

// Non-members
[wcm_nonmember]
Only non-members can see me.
[/wcm_nonmember]
```
